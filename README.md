# 📌 Tab Notes — Chrome Extension

Never forget why you opened a tab! Add sticky notes to label any tab.

## Features
- Add a note to any tab explaining why it's open
- See a 📌 badge on tabs that have notes
- View all your noted tabs in one panel
- All data stored locally — no tracking

## How to Install

1. Click the green **Code** button → **Download ZIP**
2. Unzip the folder
3. Open Chrome → go to `chrome://extensions/`
4. Enable **Developer Mode** (top right toggle)
5. Click **Load unpacked** → select the unzipped folder
